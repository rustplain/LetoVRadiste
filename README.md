# Лето в Радисте

Интерактивная визуальная новелла, разработанная в рамках учебной практики.

## Обзор проекта
«Лето в Радисте» — это нелинейное повествование с элементами аркадных мини-игр, посвященное жизни и быту студентов технических специальностей. Проект включает в себя уникальные механики взаимодействия, реализованные с помощью Python-скриптов.

## Основные особенности
* **Интерактивные механики:** Встроенная мини-игра «DragRace 1993» с динамической физикой автомобилей.
* **Графический интерфейс:** Имитация интерфейса ЭЛТ-монитора 90-х годов с адаптивной цветовой индикацией (тахометр).
* **Система отношений:** Нелинейный граф сюжета, зависящий от выбора игрока и накопленных очков взаимодействия с персонажами (Мику, Саня, Серога).
* **Адаптивные концовки:** Множественные финалы с аппаратным центрированием текста и автоматическим перезапуском сессии.

## Технологический стек
* **Движок:** [Ren'Py](https://www.renpy.org/) (версия 8.x).
* **Язык разработки:** Ren'Py Script, Python 3.
* **Контроль версий:** Git.
